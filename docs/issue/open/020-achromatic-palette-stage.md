# パレットが主役になる achromatic な器の設計

## 何をやるか

UI 自体は無彩色 (グレースケール) に抑え、ユーザーが生成したパレットの色だけが画面で目立つ設計にする。

## なぜやるか

プロダクトの主役はパレット。UI が主張すると主役が霞む。
