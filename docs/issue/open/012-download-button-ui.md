# ダウンロードボタン UI

## 何をやるか

生成された Lua ファイルをダウンロードできるボタンを設置する。クリックで Blob → File Save Dialog のシンプル実装。

## なぜやるか

ユーザーが成果物を手元に持って帰れる出口。これがないと体験が完結しない。
