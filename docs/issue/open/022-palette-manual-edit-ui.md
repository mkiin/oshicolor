# パレット手動編集 UI

## 何をやるか

生成されたパレットの各色を、ユーザーが直接微調整できる UI を提供する。color picker + リアルタイムプレビュー連携。

## なぜやるか

自動生成だけでは個人の好みに完全には合わない。最後の調整は人間が直接できるべき。
