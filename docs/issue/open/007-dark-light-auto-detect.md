# ダーク/ライトテーマ自動判定ロジック

## 何をやるか

画像の支配色の明度から、ダークテーマ/ライトテーマのどちらを生成するか自動判定する。閾値ベース（例: 平均明度 < 0.5 でダーク）が最初の素直な実装。

## なぜやるか

ユーザーが手動でテーマモードを選ばなくていい。日中/夜間のテーマ切り替え運用に対応する第一歩。

- [ ] 閾値ベース判定
