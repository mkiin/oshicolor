## splitAtom

`splitAtom` ユーティリティはリスト内の各要素に対してアトムを取得する必要があるシナリオ向けに設計されています。リストを含む読み書き可能なアトムで動作し、元のリストの各アイテムに対応するアトムのリストを保持するアトムを返します。

### シグネチャ

`splitAtom` の簡略化された型シグネチャ：

```ts
type SplitAtom = <Item, Key>(
  arrayAtom: PrimitiveAtom<Array<Item>>,
  keyExtractor?: (item: Item) => Key
): Atom<Array<PrimitiveAtom<Item>>>
```

### 主な機能

1. 返されるアトムは `write` 方向にディスパッチ関数を含んでいます（v1.6.4 以降）。`remove`、`insert`、`move` などのアクションで元のアトムを変更する簡単な方法を提供します。

2. オプションの `keyExtractor` 関数を第2引数として提供して、安定性とパフォーマンスを向上させることができます。

### キー抽出器

`splitAtom` ユーティリティはキー抽出器関数を第2引数としてサポートします：

```ts
export function splitAtom<Item, Key>(
    arrAtom: WritableAtom<Item[], [Item[]], void> | Atom<Item[]>,
    keyExtractor?: (item: Item) => Key,
);
```

`keyExtractor` の重要な考慮事項：

- React の render ループ内で `splitAtom` を使用する場合、`keyExtractor` はオブジェクト等価性（浅い等価性）を維持する安定した関数である必要があります。この要件は render ループ外では適用されません。
- `keyExtractor` を提供することでアトム出力の安定性が向上します（メモ化のキャッシュヒットが増えます）。ソース配列のインデックスシフトによる不要なサブアトムの再作成を防ぎます。
- `keyExtractor` は配列内の各アイテムに対して抽出されたキーが一意であることが保証される場合にのみ使用すべきオプションの最適化です。

### 使用例

`splitAtom` の使用を示す例：

<Stackblitz id="vitejs-vite-oavdw2" file="src%2FApp.tsx" />

```tsx
import { Provider, atom, useAtom, PrimitiveAtom } from "jotai";
import { splitAtom } from "jotai/utils";
import "./styles.css";

const initialState = [
    {
        task: "help the town",
        done: false,
    },
    {
        task: "feed the dragon",
        done: false,
    },
];

const todosAtom = atom(initialState);
const todoAtomsAtom = splitAtom(todosAtom);

type TodoType = (typeof initialState)[number];

const TodoItem = ({
    todoAtom,
    remove,
}: {
    todoAtom: PrimitiveAtom<TodoType>;
    remove: () => void;
}) => {
    const [todo, setTodo] = useAtom(todoAtom);
    return (
        <div>
            <input
                value={todo.task}
                onChange={(e) => {
                    setTodo((oldValue) => ({
                        ...oldValue,
                        task: e.target.value,
                    }));
                }}
            />
            <input
                type="checkbox"
                checked={todo.done}
                onChange={() => {
                    setTodo((oldValue) => ({
                        ...oldValue,
                        done: !oldValue.done,
                    }));
                }}
            />
            <button onClick={remove}>remove</button>
        </div>
    );
};

const TodoList = () => {
    const [todoAtoms, dispatch] = useAtom(todoAtomsAtom);
    return (
        <ul>
            {todoAtoms.map((todoAtom) => (
                <TodoItem
                    todoAtom={todoAtom}
                    remove={() => dispatch({ type: "remove", atom: todoAtom })}
                />
            ))}
        </ul>
    );
};

const App = () => (
    <Provider>
        <TodoList />
    </Provider>
);

export default App;
```

この例では、`splitAtom` を使用して todo アイテムのリストを管理し、全体のリストアトムを維持しながら各アイテムを個別に操作する方法を示しています。
