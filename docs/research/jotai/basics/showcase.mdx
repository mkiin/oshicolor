- Todo リストの例 [![Open in CodeSandbox](https://img.shields.io/badge/Open%20in-CodeSandbox-blue?style=flat-square&logo=codesandbox)](https://githubbox.com/pmndrs/jotai/tree/main/examples/todos)

    このアプリに入力して Todo リストを記録し、完了したものはチェックを入れ、`Completed` と `Incompleted` を切り替えてタスクのステータスを確認します。

- atomFamily と localStorage を使った Todo の例 [![Open in CodeSandbox](https://img.shields.io/badge/Open%20in-CodeSandbox-blue?style=flat-square&logo=codesandbox)](https://githubbox.com/pmndrs/jotai/tree/main/examples/todos_with_atomFamily)

    atomFamily と localStorage を使って Todo リストを実装します。`Save to localStorage` をクリックして Todo リストを localStorage に保存し、Todo リストを削除してから `Load from localStorage` をクリックして復元できます。
