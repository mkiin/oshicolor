`window.location` を扱うために、アトムを作成するいくつかの関数があります。

### インストール

この機能を使用するには `jotai-location` をインストールする必要があります。

```
npm install jotai-location
```

## atomWithLocation

```js
atomWithLocation(options): PrimitiveAtom
```

`atomWithLocation` は `window.location` にリンクする新しいアトムを作成します。

通常、アプリケーション内で `atomWithLocation` を1回だけインスタンス化すべきです。1つのインスタンスへの変更が他のインスタンスに反映されない場合があるためです。
このアトムは `window.location` オブジェクトと同期するように設計されているため、複数のインスタンスを使用すると予測不可能な動作が発生する可能性があります。

### パラメータ

**options**（オプション）: アトムの動作をカスタマイズするオプションオブジェクト

### オプション

**preloaded**（オプション）: 初期化時に location を取得しないようにプリロードされた location 値。

**replace**（オプション）: `pushState` の代わりに `replaceState` を使用することを示す boolean。

**getLocation**（オプション）: location 値を取得するカスタム関数。

**applyLocation**（オプション）: location 値を設定するカスタム関数。

**subscribe**（オプション）: location の変更をサブスクライブするカスタム関数。

### 例

```jsx
import { useAtom } from "jotai";
import { atomWithLocation } from "jotai-location";

const locationAtom = atomWithLocation();

const App = () => {
  const [loc, setLoc] = useAtom(locationAtom);
  return (
    <ul>
      <li>
        <button
          style={{
            fontWeight: loc.pathname === "/" ? "bold" : "normal",
          }}
          onClick={() => setLoc((prev) => ({ ...prev, pathname: "/" }))}
        >
          Home
        </button>
      </li>
      <li>
        <button
          style={{
            fontWeight:
              loc.pathname === "/foo" && !loc.searchParams?.get("bar") ? "bold" : "normal",
          }}
          onClick={() =>
            setLoc((prev) => ({
              ...prev,
              pathname: "/foo",
              searchParams: new URLSearchParams(),
            }))
          }
        >
          Foo
        </button>
      </li>
      <li>
        <button
          style={{
            fontWeight:
              loc.pathname === "/foo" && loc.searchParams?.get("bar") === "1" ? "bold" : "normal",
          }}
          onClick={() =>
            setLoc((prev) => ({
              ...prev,
              pathname: "/foo",
              searchParams: new URLSearchParams([["bar", "1"]]),
            }))
          }
        >
          Foo?bar=1
        </button>
      </li>
    </ul>
  );
};
```

## atomWithHash

```js
atomWithHash(key, initialValue, options): PrimitiveAtom
```

URL ハッシュに接続された新しいアトムを作成します。
ハッシュは URLSearchParams 形式である必要があります。
双方向バインディングです：アトム値を変更するとハッシュが変更され、
ハッシュを変更するとアトム値が変更されます。
この関数は DOM でのみ動作します。

### パラメータ

**key**（必須）: localStorage、sessionStorage、または AsyncStorage と状態を同期する際にキーとして使用される一意の文字列

**initialValue**（必須）: アトムの初期値

**options**（オプション）: アトムの動作をカスタマイズするオプションオブジェクト

### オプション

**serialize**（オプション）: アトム値をハッシュにシリアライズするカスタム関数。デフォルトは `JSON.stringify`。

**deserialize**（オプション）: ハッシュをアトム値にデシリアライズするカスタム関数。デフォルトは `JSON.parse`。

**subscribe**（オプション）: カスタムハッシュ変更サブスクライブ関数

**setHash**（オプション）: `replaceState` またはブラウザ側でハッシュを更新する方法を記述するカスタム関数。デフォルトは `window.location.hash = searchParams` 経由で履歴に新しいエントリをプッシュします（[jotai-location#4](https://github.com/jotaijs/jotai-location/pull/4)）

### 例

```jsx
import { useAtom } from "jotai";
import { atomWithHash } from "jotai-location";

const countAtom = atomWithHash("count", 1);

const Counter = () => {
  const [count, setCount] = useAtom(countAtom);
  return (
    <div>
      <div>count: {count}</div>
      <button onClick={() => setCount((c) => c + 1)}>+1</button>
    </div>
  );
};
```
