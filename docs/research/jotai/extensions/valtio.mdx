Jotai の状態は React に存在しますが、React 外の世界と連携できると便利な場合があります。

Valtio は値を格納し Jotai のアトムと同期するために使用できるプロキシインターフェースを提供します。

これは valtio のバニラ API のみを使用します。

### インストール

この機能を使用するには `valtio` と `jotai-valtio` をインストールする必要があります。

```
npm install valtio jotai-valtio
```

## atomWithProxy

`atomWithProxy` は valtio プロキシで新しいアトムを作成します。
双方向バインディングで、両端から値を変更できます。

```jsx
import { useAtom } from 'jotai'
import { atomWithProxy } from 'jotai-valtio'
import { proxy } from 'valtio/vanilla'

const proxyState = proxy({ count: 0 })
const stateAtom = atomWithProxy(proxyState)
const Counter = () => {
  const [state, setState] = useAtom(stateAtom)

  return (
    <>
      count: {state.count}
      <button
        onClick={() => setState((prev) => ({ ...prev, count: prev.count + 1 }))}
      >
        inc atom
      </button>
      <button
        onClick={() => {
          ++proxyState.count
        }}
      >
        inc proxy
      </button>
    </>
  )
}
```

### パラメータ

```
atomWithProxy(proxyObject, options?)
```

**proxyObject**（必須）: アトムを派生させる Valtio プロキシオブジェクト

**options.sync**（オプション）: バッチ更新を待たず同期的にアトムを更新します。[`valtio/useSnapshot`](https://github.com/pmndrs/valtio#update-synchronously) に似ています。レンダリングが増えますが、他の Jotai アトムとの同期がより保証されます。

```
atomWithProxy(proxyObject, { sync: true })
```

## mutableAtom

`mutableAtom` は自己認識型の Valtio プロキシで値をラップします。通常の js オブジェクトと同じように変更できます。

カウント値は `value` プロパティの下に格納されます。

```jsx
const countProxyAtom = mutableAtom(0)

function IncrementButton() {
  const countProxy = useAtomValue(countProxyAtom)
  return <button onClick={() => ++countProxy.value}>+</button>
}
```

### パラメータ

```js
mutableAtom(value, options?)
```

**value**（必須）: プロキシする値。

**options.proxyFn**（オプション）: カスタムプロキシ関数で `proxyFn` によるカスタマイズを可能にします。`proxy`（デフォルト）またはカスタム関数が指定できます。

### プロキシのミューテーションに関する注意

アトムの read 関数内やレンダリング中にプロキシを直接ミューテートしないように注意してください。これにより無限レンダリングループが発生する可能性があります。

```ts
const countProxyAtom = mutableAtom(0)

atom(
  (get) => {
    const countProxy = get(countProxyAtom)
    ++countProxy.value // 無限ループが発生します
  },
  (get, set) => {
    const countProxy = get(countProxyAtom)
    ++countProxy.value // これは問題ありません
  },
)
```
