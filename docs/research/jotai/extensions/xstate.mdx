Jotai の状態管理はプリミティブで柔軟ですが、
それは時に自由すぎることを意味します。
XState は状態管理のためのより良く安全な抽象化を提供する洗練されたライブラリです。

### インストール

この機能を使用するには `xstate` と `jotai-xstate` をインストールする必要があります。

```
npm install xstate jotai-xstate
```

## atomWithMachine

`atomWithMachine` は XState マシンで新しいアトムを作成します。
新しいマシンを作成するための関数 `getMachine` を受け取ります。
`getMachine` は最初の使用時に `get` 引数とともに呼び出され、
それを使って他のアトム値を読み取ることができます。

```tsx
import { useAtom } from "jotai";
import { atomWithMachine } from "jotai-xstate";
import { assign, createMachine } from "xstate";

const createEditableMachine = (value: string) =>
    createMachine<{ value: string }>({
        id: "editable",
        initial: "reading",
        context: {
            value,
        },
        states: {
            reading: {
                on: {
                    dblclick: "editing",
                },
            },
            editing: {
                on: {
                    cancel: "reading",
                    commit: {
                        target: "reading",
                        actions: assign({
                            value: (_, { value }) => value,
                        }),
                    },
                },
            },
        },
    });

const defaultTextAtom = atom("edit me");
const editableMachineAtom = atomWithMachine((get) =>
    // `get` はマシンの初期化にのみ使用できます
    createEditableMachine(get(defaultTextAtom)),
);

const Toggle = () => {
    const [state, send] = useAtom(editableMachineAtom);

    return (
        <div>
            {state.matches("reading") && (
                <strong onDoubleClick={send}>{state.context.value}</strong>
            )}
            {state.matches("editing") && (
                <input
                    autoFocus
                    defaultValue={state.context.value}
                    onBlur={(e) =>
                        send({ type: "commit", value: e.target.value })
                    }
                    onKeyDown={(e) => {
                        if (e.key === "Enter") {
                            send({ type: "commit", value: e.target.value });
                        }
                        if (e.key === "Escape") {
                            send("cancel");
                        }
                    }}
                />
            )}
            <br />
            <br />
            <div>
                Double-click to edit. Blur the input or press <code>enter</code>{" "}
                to commit. Press <code>esc</code> to cancel.
            </div>
        </div>
    );
};
```

### グローバル Provider で保存された再起動可能なマシン（provider-less モード）

マシンが最終状態に達すると、これ以上イベントを受け取れなくなります。
`atomWithMachine` がグローバルストア（別名 provider-less モード）で初期化されている場合、
再起動するには以下のようにマシンに `RESTART` イベントを送信する必要があります：

```tsx
import { RESTART } from "jotai-xstate";

const YourComponent = () => {
    const [current, send] = useAtom(yourMachineAtom);

    const isFinalState = current.matches("myFinalState");

    useEffect(() => {
        // コンポーネントのアンマウント時にグローバルに初期化されたマシンを再起動します
        return () => {
            if (isFinalState) send(RESTART);
        };
    }, [isFinalState]);
};
```
