# Controls & Hotkeys

キーボードショートカットと強力なナビゲーションツールでキャンバスを自在に操ろう。

## Controls & Hotkeys

Stitchはスピードを重視して設計されています。キーボードショートカットやキャンバスツールに慣れるほど、コンセプトから完成されたデザインまで、より速く進められるようになります。

### ショートカットへのクイックアクセス

いつでも `?` を押すと、キーボードショートカットの完全なリストが表示されます。このモーダルには、利用可能なすべてのショートカットがカテゴリ別に整理され、プラットフォーム固有の修飾キーが正しく表示されます（Macでは⌘、Windows/LinuxではCtrl）。

> **Pro Tip:** ショートカットモーダルはリファレンスガイドです。作業中にホットキーを忘れた場合は、`?` をタップして、フローを中断することなくすぐに確認できます。

## コアキーボードショートカット

これらは最も頻繁に使用するショートカットです。キーボードから手を離さず、キャンバスに集中できるように設計されています。

### Edit Actions

| アクション | Mac                     | Windows/Linux           |
| ---------- | ----------------------- | ----------------------- |
| Undo       | `⌘Z`                    | `Ctrl+Z`                |
| Redo       | `⌘⇧Z`                   | `Ctrl+Shift+Z`          |
| Copy       | `⌘C`                    | `Ctrl+C`                |
| Paste      | `⌘V`                    | `Ctrl+V`                |
| Duplicate  | `⌘D`                    | `Ctrl+D`                |
| Delete     | `Backspace` or `Delete` | `Backspace` or `Delete` |
| Select All | `⌘A`                    | `Ctrl+A`                |

**Duplicate** -- 選択したスクリーンを素早く複製します。

### Canvas Tools

これらのショートカットを使うと、キャンバスの操作モードを切り替えることができます。

| ツール           | ショートカット | 説明                                                                           |
| ---------------- | -------------- | ------------------------------------------------------------------------------ |
| Select Tool      | `V`            | スクリーンの選択と移動のためのデフォルトツール                                 |
| Pan Tool         | `H`            | ドラッグしてキャンバスを移動                                                   |
| Zoom Tool (hold) | `Z`            | クリックでズームイン、Alt+クリックでズームアウト、ドラッグで特定の領域にズーム |

### View Controls

| アクション  | ショートカット                |
| ----------- | ----------------------------- |
| Zoom In     | `+` or `=` or `⌘=` / `Ctrl+=` |
| Zoom Out    | `-` or `⌘-` / `Ctrl+-`        |
| Fit to View | `0` or `⌘1` / `Ctrl+1`        |

**Fit to View** -- すべてのコンテンツに合わせてズームします。

### Project Actions

| アクション                  | Mac  | Windows/Linux |
| --------------------------- | ---- | ------------- |
| Save                        | `⌘S` | `Ctrl+S`      |
| Download Selected Screen(s) | `⇧D` | `Shift+D`     |

## Command Palette (`⌘K`)

`⌘K`（Mac）または `Ctrl+K`（Windows/Linux）を押すと、Command Paletteが開きます。これはStitchで利用可能なすべてのアクションを検索するためのユニバーサル検索です。

Command Paletteは、現在利用可能なすべてのコマンドをコンテキストに基づいてフィルタリングして表示します。

- 現在の選択に基づいて、アクティブなコマンドが最初に表示されます
- 入力するとリストが即座にフィルタリングされます
- 矢印キーで移動し、Enterで実行します
- コマンドの右側にキーボードショートカットが表示されます

**なぜCommand Paletteを使うのか？**

- まだ知らない機能を発見できる
- マウスに手を伸ばさずにアクションを実行できる
- 現在のコンテキストで何ができるかを確認できる

> **Pro Tip:** やりたいことはわかっているが、正確なメニューの場所を思い出せない場合、`⌘K` が最速の方法です。数文字入力すれば、必要なコマンドが表示されます。

## キャンバスのナビゲーション

Stitchプロジェクトはすぐに大きくなり、キャンバス上に数十のスクリーンが広がることがあります。これらのショートカットを使えば効率的に移動できます。

### スクリーンナビゲーション

| アクション      | Mac  | Windows/Linux | 説明                             |
| --------------- | ---- | ------------- | -------------------------------- |
| Next Screen     | `⌘→` | `Ctrl+→`      | 読み順で次のスクリーンにジャンプ |
| Previous Screen | `⌘←` | `Ctrl+←`      | 前のスクリーンにジャンプ         |

これらのショートカットを使ってスクリーン間を移動すると、Stitchは自動的に以下を行います。

- 対象のスクリーンを選択
- そのスクリーンがビューに収まるようにズーム
- 最適な表示のために中央に配置

**Reading Order:** スクリーンは左から右、上から下の順序で並んでいます。ページを読むのと同じ順序です。これにより、複雑なプロジェクトでもナビゲーションが予測可能になります。

### キャンバスで迷ったら？

方向感覚を失った場合は、`⌘←` / `Ctrl+←` を押して最も近いスクリーンにスナップしてください。`0` または `⌘1` / `Ctrl+1` を押すと、ズームアウトしてプロジェクト全体を確認できます。

## Bottom Toolbar: Select, Pan, Zoom

下部ツールバーには3つのキャンバスツールが表示されます。各ツールはキャンバスとのインタラクション方法を変更します。

### Tool Modes

**Select Mode (`V`)**

- クリックしてスクリーンを選択
- ドラッグしてスクリーンを移動
- 空のキャンバス上でクリック＆ドラッグして選択矩形を作成
- これがデフォルトモードです。ほとんどの時間をここで過ごすことになります

**Pan Mode (`H`)**

- どこでもクリック＆ドラッグしてキャンバスを移動
- 大規模なプロジェクトのナビゲーションに便利
- 誤ってスクリーンを選択したり移動したりするリスクがない

**Zoom Mode (`Z`)**

- クリックしてズームイン（クリック位置を中心に）
- `Alt+Click` でズームアウト
- クリック＆ドラッグで矩形を描画。Stitchはその領域に正確にフィットするようにズーム
- `Escape` を押して進行中のズーム選択をキャンセル

### クイックアクセス: ホールドでアクティベート

クイックアクションのためにツールを恒久的に切り替える必要はありません。Stitchは、より速いワークフローのための一時的なモード切り替えをサポートしています。

**Hold `Space` for Quick Pan**

- 任意のモードで `Space` を押し続けると、一時的にパンモードに切り替わります
- ドラッグしてキャンバスを再配置
- `Space` を離すと、前のツールに戻ります

**Hold `Z` for Quick Zoom**

- `Z` を押し続けると、一時的にズームモードに切り替わります
- ズーム操作を実行（クリック、Alt+クリック、またはドラッグ）
- `Z` を離すと、前のツールに戻ります

**タップ vs ホールド**

- キー（`V`、`H`、`Z`）を**タップ**すると、そのツールに恒久的に切り替わります
- キー（`Space` または `Z`）を**ホールド**すると、一時的にアクティベートされ、離すと元に戻ります

**これが重要な理由:** メインのワークフローではSelectモードのまま、`Space` を押し続けてキャンバスをすばやく再配置し、離してそのまま選択と編集を続けることができます。余分なクリックもモードの混乱もありません。

### Visual Feedback

アクティブなツールは、一時的なモード切り替え中でも、下部ツールバーで常にハイライトされます。カーソルも現在のモードに合わせて変化します。

| モード | カーソル                                            |
| ------ | --------------------------------------------------- |
| Select | 矢印                                                |
| Pan    | 手（アイドル時は開いた手、ドラッグ時は握った拳）    |
| Zoom   | 虫眼鏡（ズームインは+、Alt押下時のズームアウトは-） |

## すべてを組み合わせる

これらのコントロールを組み合わせた典型的なワークフローは以下の通りです。

1. `⌘K` を押してCommand Paletteを開き、新しいスクリーンを生成
2. `⌘→` を押して新しいスクリーンにナビゲート（自動ズーム＆選択）
3. `V` でSelectモードに入り、スクリーンをクリック
4. `Space` を押し続けてパンし、スクリーンを完璧に配置
5. `Z` をタップして特定のコンポーネント周辺に矩形をドラッグしてズームイン
6. `⌘D` を押してスクリーンを複製
7. ショートカットを忘れたら `?` を押す

目標は、フローに留まりながら素早く動き、より速く優れたデザインにたどり着くことです。

### Advanced Tip: ショートカットの組み合わせ

多くのショートカットは組み合わせて使えます。

- 複数のスクリーンを選択し、`⇧D` を押して一度にすべてダウンロード
- Zoomモード中に `Space` を押し続けて、ツールを切り替えずに一時的にパン
- `⌘A` ですべてのスクリーンを選択し、`⌘D` でプロジェクトレイアウト全体を複製

スピードは筋肉記憶から生まれます。これらのショートカットを使えば使うほど、考えることが少なくなり、その分デザインについてより多く考えられるようになります。
