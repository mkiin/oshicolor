> `atomWithDebounce` は、状態設定をデバウンスするアトムの作成を支援します。

このユーティリティは、テキスト検索入力などに便利です。キーストロークのたびにアクションを実行するのではなく、指定した時間待機した後に**派生アトムの関数を1回だけ呼び出す**ことができます。

```tsx
import { atom, SetStateAction } from "jotai";

export default function atomWithDebounce<T>(
    initialValue: T,
    delayMilliseconds = 500,
    shouldDebounceOnReset = false,
) {
    const prevTimeoutAtom = atom<ReturnType<typeof setTimeout> | undefined>(
        undefined,
    );

    // currentValueAtom はエクスポートしないでください。このアトムを使って状態を設定すると、
    // currentValueAtom と debouncedValueAtom の間で状態の不整合が発生する可能性があります
    const _currentValueAtom = atom(initialValue);
    const isDebouncingAtom = atom(false);

    const debouncedValueAtom = atom(
        initialValue,
        (get, set, update: SetStateAction<T>) => {
            clearTimeout(get(prevTimeoutAtom));

            const prevValue = get(_currentValueAtom);
            const nextValue =
                typeof update === "function"
                    ? (update as (prev: T) => T)(prevValue)
                    : update;

            const onDebounceStart = () => {
                set(_currentValueAtom, nextValue);
                set(isDebouncingAtom, true);
            };

            const onDebounceEnd = () => {
                set(debouncedValueAtom, nextValue);
                set(isDebouncingAtom, false);
            };

            onDebounceStart();

            if (!shouldDebounceOnReset && nextValue === initialValue) {
                onDebounceEnd();
                return;
            }

            const nextTimeoutId = setTimeout(() => {
                onDebounceEnd();
            }, delayMilliseconds);

            // 必要に応じてクリアできるように、前のタイムアウトアトムを設定する
            set(prevTimeoutAtom, nextTimeoutId);
        },
    );

    // 必要に応じてタイムアウトをクリアするためにエクスポートするアトムセッター
    const clearTimeoutAtom = atom(null, (get, set, _arg) => {
        clearTimeout(get(prevTimeoutAtom));
        set(isDebouncingAtom, false);
    });

    return {
        currentValueAtom: atom((get) => get(_currentValueAtom)),
        isDebouncingAtom,
        clearTimeoutAtom,
        debouncedValueAtom,
    };
}
```

### 注意事項

このアトムは、高コストな更新によるページのインタラクションのブロックを防ぐことを主な目的とする React 18 の `useTransition` や `useDeferredValue` などの並行機能とは異なる目的を持っています。

詳しくは、こちらの GitHub ディスカッション https://github.com/reactwg/react-18/discussions/41 の **"How is it different from setTimeout?"** セクションをご覧ください。

### 使用例

以下のサンドボックスは、`debouncedValueAtom` の値に基づいて状態をフェッチする派生アトムの使用方法を示しています。

`<SearchInput>` にポケモンの名前を入力する際、1文字ずつリクエストを送信するのではなく、最後のテキスト入力から `delayMilliseconds` が経過した後にのみ GET リクエストを送信します。

これにより、サーバーへのバックエンドリクエスト数を削減できます。
