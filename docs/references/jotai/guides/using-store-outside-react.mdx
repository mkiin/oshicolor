Jotai の状態は React 内に存在しますが、React の外の世界と対話できると便利な場合があります。

## createStore

[`createStore`](../core/store.mdx#createstore) はアトムを格納するために使用できるストアインターフェースを提供します。ストアを使用することで、React の外からアトムの状態にアクセスして変更できます。

```jsx
import { atom, useAtomValue, createStore, Provider } from "jotai";

const timeAtom = atom(0);
const store = createStore();

store.set(timeAtom, (prev) => prev + 1); // アトムの値を更新
store.get(timeAtom); // アトムの値を読み取る

function Component() {
    const time = useAtomValue(timeAtom); // React 内
    return (
        <div className="App">
            <h1>{time}</h1>
        </div>
    );
}

export default function App() {
    return (
        <Provider store={store}>
            <Component />
        </Provider>
    );
}
```
