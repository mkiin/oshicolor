Jotai のアトムは変更なしで React Native アプリケーションで使用できます。
私たちの目標は常に React Native との完全な互換性を維持することです。

### 永続化

永続化機能に関しては、React Native に特有の実装が [utils バンドルの atomWithStorage 関数](../utilities/storage.mdx)に詳しく記載されています。

### パフォーマンス

Jotai をアプリで使用する際の既知の特定のオーバーヘッドはありません。一部のライブラリは実用的な使用のために格納データに追加のプロパティやメソッドを追加しますが、Jotai は異なる動作をし、短縮できないほどシンプルなものを常に操作しています。

Jotai のアトミックアーキテクチャは、ロジックとデータを分割することを推奨し、すべての[レンダリング（正確にはコミット）](https://legacy.reactjs.org/blog/2018/09/10/introducing-the-react-profiler.html#browsing-commits)を制御して最高のパフォーマンスを達成するための最高の体験を提供します。

そして、レンダリングは高速でなければならないことを常に覚えておいてください。計算ロジックは非同期アクションに分割してください。
