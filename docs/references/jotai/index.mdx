Jotai v2 ドキュメントへようこそ！Jotai のアトミックなグローバル React 状態管理へのアプローチは、シンプルな `useState` の代替から、複雑な要件を持つエンタープライズアプリケーションまで対応できます。

## 機能

- 最小限のコア API（2kb）
- 豊富なユーティリティと拡張機能
- TypeScript 指向
- Next.js、Waku、Remix、React Native に対応

## コア

Jotai は非常に最小限の API を持ち、メインの `jotai` バンドルからはわずかなエクスポートのみを公開しています。以下の4カテゴリに分類されています。

## ユーティリティ

Jotai にはさまざまな追加ユーティリティ関数を持つ `jotai/utils` バンドルも含まれています。例として `atomWithStorage` があり、localStorage への永続化やブラウザタブ間の同期機能を提供します。
