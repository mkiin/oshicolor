# R2/V7 課題

## 判明した課題

| # | 問題 | 原因 | 重要度 |
| --- | --- | --- | --- |
| 1 | V, DV/LV が似た色に寄る | 軸内に複数 hue が混在する場合、同じ hue 付近から V と 2色目が選ばれる | High |

## 詳細

### #1: 軸内の hue 多様性が活かされない

R1 の K=3 クラスタリングは hue 空間で行われるが、1つの軸に hue が離れた色が複数含まれることがある。
現在の V + DV/LV 競合選定は saturation/luma の距離のみで判断するため、同じ hue 付近の色が V と 2色目に選ばれやすい。

**解決案**: 軸内を hue で再クラスタリング（K=2）し、各サブクラスタから Vibrant target で1色ずつ抽出する。
これにより軸内の hue 多様性を反映した 2 seed が得られる。
