# oshicolor ロードマップ

## MVP 定義

**ゴール**: 画像1枚からダークテーマの Neovim Lua ファイルをダウンロードできる

### MVP に必要な最小機能

| フィーチャー | 必要なこと | 現状 |
|---|---|---|
| R1 色抽出 | 画像から Color Axes (main/sub/accent) を出力 | 動作する（V2） |
| R2 カラーマップ | Color Axes → ハイライトグループ割り当て | V10 実装あり、横串で繋ぐ必要あり |
| R4 Lua 生成 | ハイライトグループ → Lua ファイル出力 | **未着手（最優先）** |
| R5 プレビュー | 生成結果をブラウザで確認 | V1 基礎あり |

### MVP に不要（後回し）

- R3 コントラスト自動調整（Tone 差で構造的に保証済み、手動調整は後）
- ライトテーマ
- コミュニティギャラリー
- CSS / JSON / ターミナルエクスポート
- 多言語対応
- ユーザー認証

## MVP 後の方向性

MVP が動いたら、実際に自分で使ってみてフィードバックを得る。
改善は Issue ベースで優先順位をつけてから着手する。

### 候補（優先順未定）

- ライトテーマ対応
- パレット手動編集
- 複数画像からのテーマ生成
- エクスポート形式の拡充
